# Scripts

A set of shell and perl scripts, along with configuration files, designed for system administration and maintenance across various Unix-like operating systems.

## Structure

```
.
├── debian/        # Scripts for Debian-based distributions
├── openbsd/       # Scripts for OpenBSD systems
├── perl/          # Perl scripts (portable)
└── secureblue/    # Scripts for Secureblue (Fedora Atomic/rpm-ostree)
```
